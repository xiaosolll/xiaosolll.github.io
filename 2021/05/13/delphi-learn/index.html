各后缀文件意义

bpl:一种特殊的DLL文件，用于代码重用和减少可执行文件

dfm:

dcp:

dcu:

delphi的主模块文件格式为.dpr(类似C++的.sln文件),其它模块文件为.pas(类似C++的.cpp文件)。所以一个完整的程序由一个.dpr和若干个.pas组成

*.sln 不是C程序文件，而是visual studio 的solution文件，只是存储一些工程信息。
可以用文本编辑器直接打开。但是如果不是在visual studio中使用的话，没有意义.

delphi工程文件(*.dpr)

组成部分为：程序头、[uses]、主程序块。

1. 程序头

project 程序名

library 动态库名

2. [uses]

主程序块中使用到的.pas文件的名称必需列在uses,如窗口程式。

3. 主程序块

用begin 与 end 所包围起的一段代码.
